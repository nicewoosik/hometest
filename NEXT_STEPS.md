# 다음 단계 가이드

## ✅ 현재 상태

웹사이트가 표시되고 있지만 이미지가 제대로 로드되지 않았을 수 있습니다.

## 🖼️ 이미지 크롤링 실행

터미널에서 다음 명령어를 실행하세요:

```bash
cd /Users/tommyjang/ecstel-clone

# 이미지 크롤링 실행
npm run crawl-images
```

이 명령어는:
- HTML의 모든 이미지 다운로드
- CSS 배경 이미지 다운로드
- 누락된 이미지 찾아서 다운로드

## 📋 전체 크롤링 다시 실행 (권장)

모든 리소스를 완전히 가져오려면:

```bash
cd /Users/tommyjang/ecstel-clone

# 기존 파일 삭제 (선택사항)
rm -rf dist

# 전체 크롤링 실행
npm run crawl
```

## 🔍 확인 사항

크롤링 후 확인:
1. 이미지가 표시되는지
2. CSS 스타일이 제대로 적용되는지
3. JavaScript가 작동하는지
4. 모든 페이지 링크가 작동하는지

## 🚀 완료 후

크롤링이 완료되면:
```bash
npm run dev
```

브라우저에서 **http://localhost:8000** 접속하여 확인하세요!

---

## 💡 팁

- 이미지가 여전히 안 보이면 브라우저 개발자 도구(F12)에서 네트워크 탭 확인
- 404 에러가 나는 파일이 있는지 확인
- 해당 파일들을 수동으로 크롤링하거나 수정
