# 배경 이미지/아이콘 문제 해결

## 🔍 확인 방법

브라우저에서 개발자 도구를 열어서 확인하세요:

1. **브라우저에서 http://localhost:8000 접속**
2. **F12 또는 Cmd+Option+I** 눌러서 개발자 도구 열기
3. **Network 탭** 클릭
4. **페이지 새로고침 (F5)**
5. **빨간색(404 에러)이 나는 파일들 확인**

## 📋 확인할 것들

- 어떤 이미지 파일들이 404 에러가 나는지
- CSS 파일 경로가 올바른지
- 이미지 파일 경로가 올바른지

## 🛠️ 해결 방법

### 방법 1: CSS 경로 수정

HTML 파일에서 CSS 경로가 상대 경로로 되어 있을 수 있습니다.
`dist/NEW/html/index.html` 파일을 열어서 CSS 경로 확인:

```html
<!-- 현재 -->
<link rel="stylesheet" href="../css/ecs.css">

<!-- 수정 필요시 -->
<link rel="stylesheet" href="/css/ecs.css">
```

### 방법 2: 서버 설정 확인

서버가 상대 경로를 제대로 처리하는지 확인하세요.

### 방법 3: 누락된 이미지 다운로드

터미널에서:
```bash
cd /Users/tommyjang/ecstel-clone
npm run find-images
```

## 💡 빠른 확인

브라우저 개발자 도구에서:
1. **Console 탭**에서 에러 메시지 확인
2. **Network 탭**에서 실패한 파일 확인
3. 실패한 파일의 실제 경로 확인

실패한 파일 목록을 알려주시면 정확히 해결해드리겠습니다!
