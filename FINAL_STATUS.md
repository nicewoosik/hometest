## 🚀 서버 실행

터미널에서:

```bash
cd /Users/tommyjang/ecstel-clone
npm run dev
```

브라우저에서 **http://localhost:8000/NEW/html/index.html** 접속

## 📋 확인 사항

서버 실행 후:
1. 브라우저 개발자 도구(F12) 열기
2. Network 탭에서 실패한 파일 확인
3. Console 탭에서 에러 확인
