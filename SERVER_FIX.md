# 서버 실행 문제 해결

## 🔍 문제 확인

서버가 실행되지 않아서 접속이 안 되는 것 같습니다.

## 🛠️ 해결 방법

### 1단계: 서버 실행하기

터미널에서 다음 명령어를 실행하세요:

```bash
cd /Users/tommyjang/ecstel-clone

# 의존성이 없다면 먼저 설치
npm install

# 서버 실행
npm run dev
```

### 2단계: 정상 실행 확인

서버가 정상적으로 실행되면 터미널에 다음과 같이 표시됩니다:

```
============================================================
🌐 ECSTEL 클론 서버 실행 중
============================================================
서버 주소: http://localhost:8000
디렉토리: /Users/tommyjang/ecstel-clone/dist

서버를 중지하려면 Ctrl+C를 누르세요
============================================================
```

### 3단계: 브라우저 접속

터미널에 위 메시지가 보이면 브라우저에서:
- **http://localhost:8000** 접속

---

## ⚠️ 문제 해결

### 포트가 이미 사용 중인 경우

다른 포트로 실행:
```bash
PORT=8001 node serve.js
```

또는 `serve.js` 파일에서 포트 번호를 변경:
```javascript
const PORT = 8001  // 8000에서 8001로 변경
```

### 의존성이 없는 경우

```bash
npm install
npm run dev
```

### 서버가 시작되지 않는 경우

1. 에러 메시지 확인
2. `dist` 폴더가 있는지 확인:
   ```bash
   ls -la dist/
   ```
3. `dist` 폴더가 없다면 크롤링 먼저 실행:
   ```bash
   npm run crawl
   ```

---

## 🎯 빠른 해결

터미널에서 이 명령어들을 순서대로 실행:

```bash
cd /Users/tommyjang/ecstel-clone
npm install
npm run dev
```

터미널에 서버 실행 메시지가 보이면 브라우저에서 접속하세요!
