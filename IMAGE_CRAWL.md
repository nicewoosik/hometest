# 이미지 크롤링 가이드

## 🖼️ 이미지 크롤링 스크립트 추가

이미지가 제대로 다운로드되지 않는 문제를 해결하기 위해 전용 이미지 크롤링 스크립트를 만들었습니다.

## 🚀 실행 방법

터미널에서:

```bash
cd /Users/tommyjang/ecstel-clone

# 이미지 크롤링 실행
npm run crawl-images
```

## 📋 기능

이 스크립트는:
1. HTML 페이지의 모든 `<img>` 태그에서 이미지 URL 추출
2. CSS 파일의 `url()` 안의 배경 이미지 추출
3. 모든 이미지를 다운로드
4. 진행 상황 표시

## ⏱️ 예상 시간

- 약 2-5분 소요될 수 있습니다
- 이미지 개수에 따라 다릅니다

## ✅ 완료 후

이미지 크롤링이 완료되면:
- `dist/NEW/images/` 폴더에 이미지들이 저장됩니다
- CSS 배경 이미지도 다운로드됩니다
- 서버를 재시작하여 확인하세요

```bash
npm run dev
```

브라우저에서 **http://localhost:8000** 접속하여 이미지가 표시되는지 확인하세요!
